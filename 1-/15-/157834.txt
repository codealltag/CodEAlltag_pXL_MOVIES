Code ist eine Abkürzung für "Region-Code" und beschreibt eine
Regionalcodierung der DVD. 
Code 1 ist Nordamerika: USA, Canada, Mexico
Code 2 ist Europa und Japan

Normalerweise können Code-1-Disks nur in Code-1-Geräte abgespielt
werden usw. Einfach ausgedrückt, wollen die Filmstudios so versichern,

Region-Codes problemlos abgespielt werden können.
